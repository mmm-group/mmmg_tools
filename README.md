# mmmg_tools

## Requirements
* pymatgen
* numpy at build time

## Install instructions
### Conda install
```
$ conda create -n mmmg_tools -c matsci pymatgen
$ source activate mmmg_tools
$ git clone https://github.com/mmm-group/mmmg_tools; cd mmmg_tools
$ python setup.py install
```

### Pip install
```
$ pip install pymatgen 
$ source activate mmmg_tools
$ git clone https://github.com/mmm-group/mmmg_tools; cd mmmg_tools
$ python setup.py install
```
